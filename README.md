# odin-recipes
ENGLISH

About the project
-----------------------
This project is a collection of recipes serving as a foundation for learning and demonstrating web development skills.

Project goals
-----------------------
The objective is to practically apply HTML fundamentals and reinforce best practices when working with Git and GitHub.

Implemented concepts:
-----------------------
- HTML fundamentals: Practical understanding of tags, elements, attributes, and their values.
- Document structure: Creating a valid HTML skeleton.
- Content semantics: Proper use of headings and paragraphs.
- Element hierarchy: Managing parent-child and sibling relationships.
- Lists: Implementing ordered (ol) and unordered (ul) lists using li elements.
- Navigation and media: Creating links and embedding images with appropriate attributes.

Skill development
-----------------------
The project provides an opportunity for the regular use of Git tools and the GitHub platform in daily work.


POLSKI

O projekcie
-----------------------
Ten projekt to kolekcja przepisów kulinarnych, która służy jako fundament do nauki i prezentacji umiejętności z zakresu tworzenia stron internetowych.

Cele projektu
-----------------------
Celem jest praktyczne zastosowanie fundamentów HTML oraz utrwalenie dobrych praktyk przy pracy z git oraz github.

Zrealizowane zagadnienia:
-----------------------
- Fundamenty HTML: praktyczne zrozumienie tagów, elementów, atrybutów oraz ich wartości.
- Struktura dokumentu: tworzenie poprawnego szkieletu HTML.
- Semantyka treści: poprawne użycie nagłówków i paragrafów.
- Hierarchia elementów: operowanie relacjami rodzic-dziecko oraz rodzeństwo.
- Listy: implementacja list uporządkowanych (ol) i nieuporządkowanych (ul) przy użyciu       elementów li.
- Nawigacja i media: tworzenie linków oraz osadzanie obrazów z zachowaniem właściwych atrybutów.

Rozwój umiejętności
-----------------------
Projekt jest okazją do regularnego wykorzystywania narzędzi Git oraz platformy GitHub w codziennej pracy.
