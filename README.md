# odin-recipes

O projekcie
-----------------------
Ten projekt to kolekcja moich ulubionych przepisów kulinarnych, która służy jako fundament do nauki i prezentacji umiejętności z zakresu tworzenia stron internetowych.

Cele projektu
-----------------------
Celem jest praktyczne zastosowanie fundamentów HTML oraz utrwalenie dobrych praktyk przy pracy z systemami kontroli wersji.

Zrealizowane zagadnienia:
-----------------------
- Fundamenty HTML: praktyczne zrozumienie tagów, elementów, atrybutów oraz ich wartości.
- Struktura dokumentu: tworzenie poprawnego szkieletu HTML.
- Semantyka treści: poprawne użycie nagłówków i paragrafów.
- Hierarchia elementów: operowanie relacjami rodzic-dziecko oraz rodzeństwo.
- Listy: implementacja list uporządkowanych (ol) i nieuporządkowanych (ul) przy użyciu       elementów li.
- Nawigacja i media: tworzenie linków oraz osadzanie obrazów z zachowaniem właściwych atrybutów.

Rozwój umiejętności
-----------------------
Projekt jest okazją do regularnego wykorzystywania narzędzi Git oraz platformy GitHub w codziennej pracy.
